# AgriConnect — Frontend

Next.js frontend for the AgriConnect crop recommendation system.

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

> Make sure the Flask backend is running on `http://127.0.0.1:5000` before using the app.

## Build for Production

```bash
npm run build
npm start
```
